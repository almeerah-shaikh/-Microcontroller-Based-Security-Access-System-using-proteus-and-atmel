# -Microcontroller-Based-Security-Access-System-using-proteus-and-atmel
The Microcontroller-Based Security Access System is an embedded security project designed to provide a reliable and efficient digital locking mechanism using the ATmega32 microcontroller. The system uses a 4x4 keypad for passcode entry, a 16x2 LCD to display user prompts and access status, and a servo motor to simulate the door lock mechanism. When the correct password is entered, the microcontroller verifies it, displays a success message on the LCD, and rotates the servo motor to unlock the door. If an incorrect code is entered, access is denied and an error message is displayed. This project replaces traditional mechanical locks with a smarter and more secure electronic solution, contributing to SDG 11: Sustainable Cities and Communities by promoting smart safety and security systems. The implementation was carried out using Assembly Language and simulated in Proteus, demonstrating practical embedded system design, hardware interfacing, and real-time control.
